The roadmap

Four months, four weeks each. Click a card to mark it done as you publish — progress is saved locally in this browser. What changed since v1 →

What changed from v1: added a raw CUDA week (Week 6), folded real profiling tools in from that point on instead of relying only on wall-clock timing, added a numerical-correctness + mixed-precision habit that runs through every week from Week 8 onward, and added a dedicated week reading real production kernel code (Week 14).

Standing habit, starting Week 8: every notebook validates its custom implementation against a reference (torch.allclose / max-error check), and every GPU benchmark from Week 6 onward reports at least one profiler-derived number — achieved memory bandwidth or occupancy — not just wall-clock time.
